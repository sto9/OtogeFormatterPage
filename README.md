# OtogeFormatter(旧：音ゲー曲の表記揺れ直すやつ)

[OtogeFormatter](https://sto9.github.io/OtogeFormatterPage/)

## 概要

音ゲー曲の表記揺れを直すためのツール(Webサイト)です。

入力されたテキストに対し、**実際に存在する曲の存在する難易度** の中から最も近いものを選出し、統一されたフォーマットで置き換えます。

音ゲーの企画で曲を google form で集計したが、表記揺れを統一するのが面倒……という場合などに便利です。

## 対応機種

- CHUNITHM
- SOUND VOLTEX
- maimai
- オンゲキ

## 使い方

フォーマットを選んで曲名を貼り付けるだけです。1行につき1曲でお願いします。

修正後の曲名の書式で差分の有無がわかります：

- 細字・黒文字 → 差分なし
- 太字・黒文字 → スペースや括弧など、軽微な差分のみ
- 太字・赤文字 → 曲名に関する差分あり
